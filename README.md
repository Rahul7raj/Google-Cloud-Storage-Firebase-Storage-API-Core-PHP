  <h3>To generate a private key file for your service account:</h3>
  <ol>
<li><p>In the Firebase console, open
<strong>Settings &gt; <a href="//console.firebase.google.com/project/_/settings/serviceaccounts/adminsdk">Service Accounts</a></strong>.</p></li>
<li><p>Click <strong>Generate New Private Key</strong>, then confirm by clicking <strong>Generate Key</strong>.</p></li>
<li><p>Securely store the JSON file containing the key.</p></li>
</ol>
  
  <h3>life cycle for auto file deletion</h3>
  <p>If you want to auto delete files after some time then follow this 
https://cloud.google.com/storage/docs/managing-lifecycles</p>
